# De beste ChatGPT-prompt die ik ken

**Auteur:** Erik van der Veen
**Datum:** 11 juni 2025
**Leestijd:** 2 minuten

---

Stel, je wilt iets aan ChatGPT vragen. Je kunt zelf een prompt bedenken, natuurlijk. Maar er is een slimmere aanpak: laat ChatGPT de prompt *voor jou* schrijven. Het model stelt je gerichte vragen, je beantwoordt ze, en samen kom je tot een prompt die veel beter is dan wat je zelf in één keer had bedacht.

Klinkt dat te mooi? Probeer het eens. Begin een nieuw chatgesprek met de onderstaande prompt en je zult zien hoe goed dit werkt.

## De prompt

> Word mijn Prompt Maker. Je bent daarin de allerbeste ter wereld. Help me de ideale prompt voor mijn behoeften te creëren via onderstaande structuur.
>
> 1. Vraag naar het onderwerp van de prompt.
> 2. Geef op basis van mijn antwoord:
>    a) **Aangepaste prompt** \u2013 een duidelijke, beknopte versie.
>    b) **Suggesties** \u2013 ideeën om de prompt te verbeteren.
>    c) **Vragen** \u2013 wat je verder nodig hebt om de prompt te verfijnen.
> 3. Werk de prompt bij volgens mijn feedback.
> 4. Beoordeel elke prompt op een schaal van 1\u201310. Is de score 8 of hoger, vraag dan of ik deze wil uitvoeren met een "Ja / Nee" keuze. Voer de prompt uit als ik "Ja" zeg. Wees kritisch in je beoordeling.
> 5. Herhaal deze stappen totdat ik aangeef te stoppen of de prompt te willen uitvoeren.

## Waarom werkt dit zo goed?

De kracht zit in het iteratieve proces. In plaats van één enkele poging doe je meerdere rondes van verfijning. ChatGPT kent de structuur van een goede prompt beter dan de meeste gebruikers, dus door het model te laten meedenken over de opzet, kom je tot resultaten die je zelf niet zo snel had bereikt.

Daarnaast zorgt de beoordelingsschaal ervoor dat ChatGPT kritisch blijft. Het model voert de prompt pas uit als de kwaliteit hoog genoeg is. Dat voorkomt dat je genoegen neemt met een matige eerste versie.

## Wanneer gebruik je dit?

Deze aanpak is vooral waardevol bij complexere opdrachten: een strategisch plan, een uitgebreide analyse, een goed gestructureerde presentatie of een creatief concept. Voor simpele vragen ("Wat is de hoofdstad van Australië?") is het uiteraard overkill.

## Probeer het zelf

Open een nieuw gesprek, plak de prompt erin, en geef als onderwerp iets op waar je op dit moment mee bezig bent. Je zult verrast zijn door het verschil.
